Guiamiento, Navegación y Control (GNC)

El sistema de Guiamiento, Navegación y Control (GNC) del cohete JICAAT V es el encargado de estimar, supervisar y corregir la orientación, posición y trayectoria del vehículo durante todas las fases del vuelo: despegue, ascenso, vuelo balístico y recuperación.

Para la estimación de actitud y navegación, el sistema integra sensores inerciales (IMU con acelerómetros y giróscopos), receptor GPS y sensores ambientales, combinados mediante algoritmos de fusión de sensores que permiten obtener datos confiables de posición, velocidad y orientación en tiempo real.

El subsistema de control implementa algoritmos de control experimental, incluyendo control PID y esquemas predictivos simplificados, los cuales actúan sobre las superficies de control y/o sistemas de estabilización para minimizar perturbaciones aerodinámicas, vibraciones estructurales y errores de trayectoria.

El GNC del JICAAT V ha sido diseñado como una plataforma modular y escalable, permitiendo la validación de nuevas estrategias de control, pruebas de software en vuelo y la futura incorporación de algoritmos avanzados de control autónomo y navegación de alta precisión.

Funcionalidad Sistema de Guiado y Control para Vehículo Lanzador

En el caso del vehículo lanzador JICAAT, el sistema desarrollado por Villa Automation SAC cumple las siguientes funciones específicas: • Determinación autónoma de trayectoria mediante simulación en Scilab/Xcos. • Estimación inercial mediante sensores IMU-GPS integrados. • Ejecución de control PID/LQR sobre actuadores de empuje vectorial. • Transmisión de telemetría a estación base por radiofrecuencia. Registro de parámetros de vuelo (altitud, orientación, aceleración) en memoria SD para análisis post-lanzamiento. Estas funcionalidades permiten validar los algoritmos de guiado y control desarrollados, así como demostrar la capacidad nacional de implementar sistemas autónomos de precisión aplicables a la defensa y la investigación aeroespacial, como se muestra la figura.
