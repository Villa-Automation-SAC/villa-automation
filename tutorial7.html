1) ¿Qué es Proteus?

Proteus es un software que permite simular circuitos electrónicos y ejecutar código de Arduino sin necesidad de tener el hardware físico.

Es ideal para pruebas rápidas, aprender electrónica y validar proyectos antes de implementarlos.

Aquí puedes descargar el instalador o usar tu versión actual.

⬇ Descargar Proteus

New Project → Schematic Capture

3) Simular un LED en Proteus

Componentes necesarios:

ir a tienda ->
  • Arduino UNO
  • LED rojo
  • Resistencia 220Ω
  • Cables de conexión

Usa el siguiente código en tu Arduino:

void setup() {
  pinMode(3, OUTPUT);
}

void loop() {
  digitalWrite(3, HIGH);
  delay(1000);
  digitalWrite(3, LOW);
  delay(1000);
}

4) Cargar archivo HEX de Arduino en Proteus

En Arduino IDE haz:

  1. Sketch → Exportar compilado binario
  2. Ve a la carpeta del proyecto y ubica el archivo .hex

En Proteus:

  • Doble clic en la placa Arduino
  • En Program File → selecciona tu archivo .hex

5) Proyecto completo en Proteus

Control de brillo de LED con potenciómetro.

int entrada = A0;
int led = 9;

void setup() { pinMode(led, OUTPUT); }

void loop() {
  int valor = analogRead(entrada);
  int brillo = map(valor, 0, 1023, 0, 255);
  analogWrite(led, brillo);
}

Simula moviendo el potenciómetro y verás el brillo variar.
