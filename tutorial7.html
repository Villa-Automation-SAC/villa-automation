7) Tutorial Proteus — Simular LED con Arduino

En este tutorial aprenderás a simular un circuito básico en Proteus usando Arduino UNO y un LED conectado al pin 3.

Objetivo: Simular el código Blink dentro de Proteus para verificar el funcionamiento sin usar hardware real.

Materiales en Proteus

  • Arduino UNO (modelo: Arduino UNO R3)
  • LED rojo
  • Resistencia 220 Ω
  • Cables de conexión

Conexiones

Conecta los elementos así:

  • Pin digital 3 → resistencia de 220 Ω → LED (ánodo)
  • Cátodo del LED → GND
  • Arduino alimentado automáticamente por Proteus

Cargar el código en Proteus

Compila tu código en Arduino IDE y genera el archivo .hex:

  1. Ve a Sketch → Exportar binarios compilados.
  2. Ubica el archivo: carpeta del proyecto → .ino.hex.
  3. En Proteus, da doble clic al Arduino y carga el archivo .hex.

Código usado en Proteus

// Enciende y apaga LED en el pin 3 (para usar en Proteus)
void setup() {
  pinMode(3, OUTPUT);
}

void loop() {
  digitalWrite(3, HIGH);
  delay(1000);
  digitalWrite(3, LOW);
  delay(1000);
}

Ahora presiona el botón ▶ RUN en Proteus y observarás el LED parpadear igual que en la placa real.
